## Project structure :

- __src__
  - __assets__
    - __fonts__ (font)
    - __images__ (all images)
  - __js__ (javascript/libs)
  - __pug__ (pages)
    - __parts__ (components)
  - __scss__ (styles)
- __dist__
  - __assets__
    - __fonts__
    - __images__
  - __parts__
 

run in local with : ```npm run dev``` \n
build html rendered with : ```npm run build```
